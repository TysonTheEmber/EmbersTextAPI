# Command Demos

| Command | Purpose |
| --- | --- |
| `/emberstextapi test <id>` | Play built-in demonstration messages. |
| `/emberstextapi send <player> <duration> [fadeIn] [fadeOut] <text>` | Send a basic message with optional fade timings. |
| `/immersivemessages sendcustom @p {text:"Boss Approaches!",duration:60,fadeIn:10,fadeOut:20}` | Demonstrates a fade-in/out custom message. |
