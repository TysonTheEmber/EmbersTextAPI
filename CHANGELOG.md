# Changelog

- Added per-message fadeIn/fadeOut tick attributes that wrap around duration.
- New builder methods: fadeInTicks(int), fadeOutTicks(int)
- Commands now accept NBT keys `fadeIn`, `fadeOut` (ints).
- Added Ember Markup tag parser, vanilla component emitter, and overlay hooks.
- Introduced attribute registry with built-in color/gradient/wave/shake handlers.
- Added `/emberstextapi preview <markup>` client command for quick testing.
